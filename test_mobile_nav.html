üìè Responsive Testing Instructions

Method 1: Browser DevTools

  1. Press F12 to open DevTools
  2. Press Ctrl+Shift+M (or Cmd+Shift+M on Mac) for device toolbar
  3. Select different devices from dropdown
  4. Test hamburger menu on each device size

Method 2: Manual Resize

  1. Resize browser window to less than 992px wide
  2. Hamburger menu should appear
  3. Resize back to 992px or wider
  4. Hamburger menu should disappear
